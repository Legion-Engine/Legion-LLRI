struct llri::device_desc¶

Device description to be used in Instance::createDevice().

Public Members

Adapter *adapter¶

The adapter to create the instance for.

adapter_features features¶

The enabled adapter features. It is recommended to only enable features that will be used because unused enabled features might disable driver optimizations.

uint32_t numExtensions¶

The number of device extensions in the device_desc::extensions array.

adapter_extension *extensions¶

The adapter extensions, if device_desc::numExtensions > 0, then this must be a valid pointer to an array of adapter_extension. If device_desc::numExtensions == 0, then this pointer may be nullptr.
