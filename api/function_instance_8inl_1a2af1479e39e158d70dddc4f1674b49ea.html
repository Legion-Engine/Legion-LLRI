Function llri::queryInstanceExtensionSupport¶

  • Defined in File instance.inl

Function Documentation¶

inline bool llri::queryInstanceExtensionSupport(instance_extension_type type)¶

Queries the support of the given extension type. Support for an extension may depend on device configuration, hardware compatibility, or other environment factors. The support for extensions may differ between implementations.

Returns

true if the extension is supported, and false if it isn’t.
