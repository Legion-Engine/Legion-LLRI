Class Documentation¶

class llri::Instance¶

Instance is the central resource of the application and is used to create most other API objects. Only a single instance of Instance may exist within an application. Creating more instances is not officially supported.

Public Functions

inline result enumerateAdapters(std::vector<Adapter*> *adapters)¶

Retrieve a vector of adapters available to this application. Adapters usually represent PCIe devices such as GPUs.

Using this function, you can select one or more adapters for Device creation. The adapters returned by this function are individual adapters and are listed separately regardless of SLI/Crossfire/Multi-GPU configuration.

Parameters

adapters – The vector to fill with available adapters. The vector is cleared at the start of the operation and is only filled if the operation succeeds.

Returns

Success upon correct execution of the operation.

Returns

ErrorInvalidUsage if adapters is nullptr.

Returns

Implementation defined result values: ErrorOutOfHostMemory, ErrorOutOfDeviceMemory, ErrorInitializationFailed.

inline result createDevice(const device_desc &desc, Device **device) const¶

Creates a virtual LLRI device. Device represents one or multiple adapters and is used to allocate memory, create resources, or send commands to the adapter.

Parameters

  • desc – The description of the device.

  • device – A pointer to a device pointer variable. The pointer variable will be set to the resulting device upon successful execution.

Returns

Success upon correct execution of the operation.

Returns

ErrorInvalidUsage if the instance is nullptr, if device is nullptr, if desc.adapter is nullptr, or if desc.numExtensions is more than 0 and desc.extensions is nullptr.

Returns

ErrorDeviceLost if the adapter was lost.

inline void destroyDevice(Device *device) const¶

Destroy the LLRI device.

All resources created through the device must be destroyed prior to calling this function.

Parameters

device – The device, must be a valid device pointer or nullptr.
