Struct Documentation¶

struct llri::instance_desc¶

Instance description to be used in llri::createInstance().

Public Members

uint32_t numExtensions¶

The number of instance extensions in the instance_desc::extensions array.

instance_extension *extensions¶

The instance extensions, if instance_desc::numExtensions > 0, then this must be a valid pointer to an array of instance_extension. If instance_desc::numExtensions == 0, then this pointer may be nullptr.

const char *applicationName¶

Sets the name of the application in the implementation if applicable.

Note

This parameter is not guaranteed to be used but is known to at least apply to Vulkan.

validation_callback_desc callbackDesc¶

Describes the optional validation callback. callbackDesc.callback may be nullptr in which case no callbacks will be sent.
