Function llri::createInstance¶

  • Defined in File instance.inl

Function Documentation¶

inline result llri::createInstance(const instance_desc &desc, Instance **instance)¶

Create an llri Instance. Like with all API objects, the user is responsible for destroying the Instance again using destroyInstance().

Parameters

  • desc – The description of the instance.

  • instance – Must be a valid pointer to an Instance pointer variable. Upon successful execution of the operation the pointer is set to the resulting instance object.

Returns

Success upon correct execution of the operation.

Returns

ErrorInvalidUsage if the instance is nullptr, or if desc.numExtensions > 0 and desc.extensions is nullptr.

Returns

ErrorExtensionNotSupported if any of the extensions fail to be created.

Returns

Implementation defined result values: ErrorExtensionNotSupported, ErrorOutOfHostMemory, ErrorOutOfDeviceMemory, ErrorInitializationFailed, ErrorIncompatibleDriver.
