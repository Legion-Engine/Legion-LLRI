Function llri::destroyInstance¶

  • Defined in File instance.inl

Function Documentation¶

inline void llri::destroyInstance(Instance *instance)¶

Destroys the given instance and its directly related internal resources. All resources created through the instance must be destroyed prior to calling this function.

Parameters

instance – The instance to destroy. This value must be a valid instance pointer or nullptr.
