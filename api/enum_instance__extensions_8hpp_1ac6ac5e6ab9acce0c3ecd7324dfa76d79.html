enum llri::instance_extension_type¶

Describes the kind of instance extension. This value is used in instance_extension and is used in createInstance() to recognize which value to pick from instance_extension’s union.

The support for each available instance_extension is fully optional (hence their name, extension), so it is recommended to use this enum with queryInstanceExtensionSupport() to find out if a desired extension is available prior to adding the extension to the instance_desc’s extensions array.

Values:

enumerator DriverValidation¶

Validate operations on a driver level. Driver validation often does additional parameter and context checks for the implementation’s operations.

enumerator GPUValidation¶

Validate shader operations such as buffer reads/writes. This kind of validation heavily injects itself into shaders, causing a potentially significant performance impact.

enumerator MaxEnum¶

The highest value in this enum.
