using llri::validation_callback = void(validation_callback_severity severity, validation_callback_source source, const char *message, void *userData)¶

The validation callback function. The callback passes numerous parameters which help classify the message’s severity and source. It also passes the userData pointer that was initially passed in validation_callback_desc.
