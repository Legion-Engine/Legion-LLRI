Class Documentation¶

class llri::Adapter¶

Represents a compatible adapter (GPU, APU, IGPU, etc.). This handle is created and owned by the Instance, the user is not responsible for destroying it.

Public Functions

inline result queryInfo(adapter_info *info) const¶

Query basic information about the Adapter.

Parameters

info – A pointer to the adapter_info structure that needs to be filled.

Returns

ErrorInvalidUsage if info is nullptr.

Returns

ErrorDeviceLost If the adapter was removed or lost.

inline result queryFeatures(adapter_features *features) const¶

Query a structure with all supported driver/hardware features.

Parameters

features – A pointer to the adapter_features structure that needs to be filled.

Returns

Success upon correct execution of the operation.

Returns

ErrorInvalidUsage if features is nullptr.

Returns

ErrorIncompatibleDriver if the Adapter doesn’t support the implementation’s requested graphics API.

Returns

ErrorDeviceLost If the adapter was removed or lost.

inline result queryExtensionSupport(adapter_extension_type type, bool *supported) const¶

Query the support of a given adapter extension.

Parameters

  • type – The type of adapter extension to check against.

  • supported – A pointer to the boolean that describes if the extension is supported.

Returns

Success upon correct execution of the operation.

Returns

ErrorInvalidUsage If supported is nullptr.

Returns

ErrorDeviceLost If the adapter was removed or lost.
