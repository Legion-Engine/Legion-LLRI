enum llri::validation_callback_source¶

Describes the source of the validation callback message.

Values:

enumerator Validation¶

The message came from the LLRI API. API validation does basic parameter checks to make sure that the API doesn’t crash.

Note

This value never occurs if LLRI_DISABLE_VALIDATION is defined.

enumerator Implementation¶

The message came from the implementation. Implementation validation needs to be enabled through driver_validation_ext and/or gpu_validation_ext for this kind of message to appear.

Note

This value never occurs if LLRI_DISABLE_IMPLEMENTATION_MESSAGE_POLLING is defined.

enumerator MaxEnum¶

The highest value in this enum.
